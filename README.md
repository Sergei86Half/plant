# Go Plant Grow Simulator 🌱

Минималистичный текстовый симулятор, в котором вы ухаживаете за растением, помогая ему пройти путь от маленького семечка до прекрасного подсолнуха.

## Как играть?
1. **Запустите игру** через терминал.
2. **Поливайте растение**, нажимая клавишу `ENTER`.
3. **Следите за прогрессом**: каждые 3 полива растение переходит на новую стадию роста.
4. **Цель**: достичь финальной стадии «Полностью выросшее! 🌻».

## Стадии роста
1. 🌱 Семечко
2. 🌿 Молодое растение
3. 🌳 Взрослое растение
4. 🌻 Полностью выросшее!

## Технические особенности
- Написано на **чистом Go** без внешних зависимостей.
- Используется объектно-ориентированный подход (структура `Plant` и методы).
- Интерактивный консольный интерфейс с использованием `bufio`.

## Запуск

go run main.go
